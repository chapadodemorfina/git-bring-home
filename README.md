# i9 — Sistema de Gestão Técnica

Sistema completo de gestão para assistência técnica, incluindo controle de ordens de serviço, estoque, logística, financeiro, portal do cliente e muito mais.

## Empresa

**i9 Solution**

## Desenvolvido por

**Alvo Sistemas e Gestão**

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase (backend)

## Instalação

```sh
git clone <URL_DO_REPOSITORIO>
cd <NOME_DO_PROJETO>
npm install
npm run dev
```

## Deploy

O sistema é hospedado via Cloudflare Pages conectado ao GitHub.
